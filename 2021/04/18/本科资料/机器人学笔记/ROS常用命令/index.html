ROS常用命令

发表于2021-04-18|更新于2025-03-03
|浏览量:

  • 1.rosrun packegename nodename; 启动功能包中可执行文件/节点

  • 2.roscore 启动rosmaster

  • 3.rosnode 节点相关功能

  • 4.rostopic 话题相关功能

  • 5.rosservice 服务相关功能

  • 6.rosmsg 消息相关

  • 7.rossrv

  • 8.catkin_create_pkg

  • 9.catkin_make

  • 10.rospack find/list 获取功能包信息

  • 11.roslaunch 运行launch文件

  • 12.rqt 图形系统

  • 13.catkin_init_workspace

  • 14.rosparam 参数

检查环境变量：echo $ROS_PACKAGE_PATH

编译特定软件包：catkin_make -DCATKIN_WHITELIST_PACKAGES=“包名”

生成install：catkin_make install

创建功能包：catkin_create_pkg [depend1] [depend2] [depend

4.添加程序包到全局路径

$ echo “source catkin_ws/devel/setup.bash” >> ~/.bashrc
$ source ~/.bashrc

5.查看 Package 依赖

$ rospack depends
$ rospack depends1
